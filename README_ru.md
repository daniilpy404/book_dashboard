# 📊 Book Dashboard

Интерактивный дашборд на Streamlit для анализа данных о книгах, собранных с помощью веб-скрейпинга. Проект включает интеграцию с PostgreSQL, фильтрацию по жанрам и удобный интерфейс для визуализации данных.

---

## 🚀 Возможности

- 📚 Отображение данных о книгах (название, жанр, цена, ссылка)
- 🎯 Фильтрация по жанрам
- 🔎 Удобный интерфейс для анализа данных
- ⚡ Быстрый и лёгкий UI на Streamlit
- 🛢 Интеграция с PostgreSQL

## ⚙️ Установка

### 1. Клонировать репозиторий


git clone https://github.com/yourusername/book_dashboard.git
cd book_dashboard


### 2. Установить зависимости


pip install -r requirements.txt


---

## 🛢 Настройка базы данных

Убедись, что PostgreSQL установлен и запущен.

Создай базу данных:

```sql
CREATE DATABASE books_db;

Обнови данные в config.py:

DB_CONFIG = {
    "dbname": "books_db",
    "user": "postgres",
    "password": "your_password",
    "host": "localhost",
    "port": "5432"
}
▶️ Запуск

Запусти приложение:

streamlit run app.py

Открой в браузере:

http://localhost:8501
